A Mini Project With a Real Purpose

Screenova was created as a second-year engineering mini project by four students who noticed a simple truth: everyone around them was struggling with phone addiction, yet no simple, private tool existed to track it.

We built Screenova to be exactly that — a clean, honest mirror for your digital habits. No algorithms that manipulate you, no cloud servers storing your data, no subscription fees.

🔒
Privacy First
All data lives in JSON files on your own machine. Nothing is uploaded to any server.
🎓
Built for Students
Simple Flask backend, plain HTML/CSS/JS — every line of code is readable and learnable.
💚
Genuinely Helpful
Not just a tracker — real detox tools, wellness tips, and email nudges that create lasting change.

Make sure you added exactly "http://127.0.0.1:5000/google/callback" as an Authorised Redirect URI in your Google Cloud Console project. Also add yourself as a Test User in the OAuth consent screen.
